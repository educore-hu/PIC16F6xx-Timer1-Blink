# PIC16F6xx-Blink

Blink (LED villogtató) program az Educore-hu által forgalmazott áramkörhöz.

## Leírás
Ez a projekt egy egyszerű példaprogram, amely egy LED-et villogtat a mikrokontroller Timer1 használatával. Ideális az áramkör és a fejlesztőkörnyezet első teszteléséhez,
és a megszakításkezelés tanulásához.

### Tesztelt környezet
* **Mikrokontroller:** PIC16F690
* **Hardver:** Educore-hu saját fejlesztésű áramköre
* **Fejlesztőkörnyezet:** MPLAB X v5.00 / XC8 fordító

## Használat
1. Nyisd meg a projektet a fejlesztőkörnyezetedben.
2. Ellenőrizd a konfigurációs biteket a forráskódban.
3. Fordítsd le és égesd a programot a PIC16F690-es mikrokontrollerre.
4. Az áramkör bekapcsolása után a LED-nek villognia kell.
